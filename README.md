## Installation

1. Install theme from the [Marketplace](https://marketplace.visualstudio.com/items?itemName=MehdiTalalha.zene-theme)
2. Go to `File > Preferences > Color Theme`
3. Select `Zene Dark Theme` or `Zene Light Theme`

Alternatively install via CLI:
```
code --install-extension MehdiTalalha.zene-theme
```
